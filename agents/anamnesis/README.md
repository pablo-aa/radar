# Anamnesis Agent

Trigger: on sign-up + quarterly re-run. Input: GitHub username + experience doc (CV-style PDF, signal only for skills and history extraction) + personal site. Tools: web_fetch (read personal site), bash (parse PDF), custom "save_profile" tool. Output: structured profile JSON to the profiles table.

Anamnesis ingests a CV-style document as a source of skills and project history. It does not do resume optimization, rewriting, or tuning. Radar's scope is grants, fellowships, bolsas, accelerators, and arenas; no resume or job-matching products.
